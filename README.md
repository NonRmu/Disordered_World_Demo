# Disordered_World

## 一、概述

- 核心玩法为在场景中对“物体存在状态”进行编辑，分为以下三种类型：

  - 物理实体 / Solid

    - 受物理影响且有碰撞；
    - 可用于角色站立、推动或触发机关等；
    - 为可编辑物体的默认存在状态；

  - 字符虚体 / Virtual

    - 无法直接移动且无碰撞；
    - 由玩家主动创造的虚体存在一定上限；
    - 尝试转换实体时若自身位置有碰撞则会被销毁；
    - 为可编辑物体的过渡状态；

  - 平面投影 / Projection

    - 为物体在屏幕空间下的投影；
    - 可在投影模式下被移动；
    - 与其他平面投影在特定角度连通时可一起移动；
    - 可移动其完全覆盖的物理实体；

- 玩家拥有四种能力：

  - 字符枪：可进入瞄准模式，用于将Solid转换为Virtual；
  - 扫描仪：可进入观测模式，用于将Virtual转换为Solid；
  - 投影仪：可进入投影模式，用于将Virtual转换为Projection；
  - 回溯器：可进入回溯模式，用于重置一定范围内的所有可编辑物体；

- 游戏初始为训练关卡，按Esc打开菜单可进行关卡选择；
- 游戏目标为进入场景内的红色特殊区域，由此切换至下一关。

---

## 二、操作规则

### 1. 基础操作

- W/A/S/D：角色移动

- Space：角色跳跃

  - 添加了落地预输入和土狼跳功能；

- 鼠标移动：用于控制摄像机旋转

- 鼠标左键：用于确认目标、选择、拖拽

- 鼠标右键：长按进入瞄准模式

  - 准星对准实体并点按鼠标左键可转换目标实体；

- E：点按进入或退出投影模式

  - 准星对准投影并长按鼠标左键可移动目标投影；

- Q：点按进入观测模式

  - 倒计时期间再次点按可将一定范围内的虚体转换为实体；

- R：长按进入回溯模式

  - 分为两段：第一段用于确认是否可使用，第二段用于位置预览；

- Esc：点按显示或隐藏暂停菜单

- Tab：点按显示或隐藏调试用RenderTexture

---

### 2. 核心能力

#### 2.1 瞄准模式

- 长按鼠标右键时进入Aim模式，可将Solid转换为Virtual；
- 从屏幕中心发出射线检测目标Layer的物体，符合条件则将其转换；
- 玩家创造的Virtual存在一定数量上限，超出数量则回退最开始的物体；
- 若Virtual碰撞阈值范围内存在目标LayerMask的物体，则回退时销毁自身；

---

#### 2.2 观测模式

- 点按Q键进入观测模式，可将Virtual转换为Solid；
- 第一次点按Q键会进入倒计时并进入预览状态，倒计时结束后退出观测模式；
- 预览状态下会以高亮显示框的方式显示以角色为中心一定范围内的Virtual；
- 倒计时期间再次点按Q键，若此时范围内存在可转换物体则将其转换；
- 若Virtual碰撞阈值范围内存在目标LayerMask的物体，则转换时销毁自身；

---

#### 2.3 投影模式

- 点按E键进入或退出投影模式，可将Virtual转换为Projection；
- 投影模式下屏幕中心会出现投影判定区域，区域内的Virtual会变成Projection；
- Virtual的判定会以Bounds的形式，根据采样比例阈值进行判定；
- 屏幕中心对准目标Projection点击鼠标左键时则通过RT用8邻域像素连通判断相连投影组；
- 若投影组在屏幕空间下完全遮挡Solid物体，则可带动其一起移动；
- 此时按住鼠标左键不放可同时移动投影组的所有物体和完全遮挡的所有Solid物体；
- Projection存在多种状态：

  - 普通投影状态 / Projection

  - 有覆盖Solid时拖拽，二者都正常的状态 / Projection_Dragging

  - 有覆盖Solid时拖拽，Solid非法时的状态 / Projection_Invalid

    - 若Solid阈值范围内有目标LayerMask物体则进入该状态；
    - 此时松开鼠标左键Solid将会恢复拖拽时的初始位置，Projection位置不变；

  - 无覆盖Solid时拖拽，Projection非法时的状态 / Projection_Invalid_WithSolid

    - 若Projection阈值范围内有目标LayerMask物体或松手时被完全遮挡则进入该状态；
    - 此时的Projection_Invalid_WithSolid不参与连通判断，满足条件后可恢复Projection；

---

#### 2.4 回溯模式

- 长按R键进入回溯模式，可重置一定范围内的所有Solid/Virtual/Projection；
- 长按R键共分为两段：第一段用于确认是否可使用，第二段用于位置预览；
- 预览状态下会以Virtual的形式显示可编辑物体原先的位置和旋转；
- 预览状态下取消不进入CD，第二段结束后进入CD；
- 若预览时Virtual碰撞阈值范围内存在目标LayerMask的物体，则回溯后销毁自身；
